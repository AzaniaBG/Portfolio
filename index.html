Tech Used: HTML:5, CSS3, JavaScript, jQuery. View Quiz App Repository

Project Description

This Spanish Quiz App tests for intermediate-level Spanish language knowledge. Five questions require the quizee choose the correct translation. Users can view:
  1. 1. The question they are on;
  2. 2. How many questions they've answered correctly;
  3. 3. Feedback on each answer and
  4. 4. Their final score.
